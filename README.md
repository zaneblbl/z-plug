## BookMark
书签同步功能
```
    同步书签到github
        github配置：1.UserName:用户名
                    2.Access Token:替代密码的登录令牌（github Settings -> Developer settings -> Personal access tokens ）
                    3.Path:书签保存的Json路径（需先创建项目，创建Json文件）
                    4.SaveMessage:是否保存登录信息
```
